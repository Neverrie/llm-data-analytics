# Backend (FastAPI)

Backend-часть учебного проекта LLM Data Analyst Lab.

## Запуск через Docker Compose (из корня репозитория)

```bash
docker compose up --build
```

## Доступные endpoint'ы

- `GET /api/health`
- `GET /api/lab1/status`
- `GET /api/lab2/demo`
- `GET /api/lab3/status`

Swagger:

- `http://localhost:8000/docs`
